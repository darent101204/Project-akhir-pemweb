# Project Akhir Pemrograman Web

This project is a final assignment for the **Web Programming** course.

## Project Description
This application is a coffee-themed web application built using **Laravel**.  
The project was initially developed as a static website (HTML, CSS, JavaScript) and later migrated into Laravel using Blade templates.

## Features
- Coffee-themed landing page
- Product management (CRUD)
- MySQL database integration
- Search functionality
- Responsive UI with interactive elements
- MVC architecture using Laravel

## Technologies Used
- Laravel
- MySQL
- Blade Template Engine
- HTML, CSS, JavaScript
